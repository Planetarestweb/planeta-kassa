# 🪐 Касса Планета

Веб-приложение для кассира детского клуба. Работает в браузере, данные хранятся на сервере, синхронизируется между устройствами.

---

## Как запустить (бесплатно на Render.com)

### Шаг 1 — Загрузи код на GitHub

1. Зайди на [github.com](https://github.com) → зарегистрируйся (если нет аккаунта)
2. Нажми **New repository** → назови `planeta-kassa` → Create
3. Загрузи все файлы из этой папки в репозиторий

### Шаг 2 — Задеплой на Render

1. Зайди на [render.com](https://render.com) → Sign Up (через GitHub)
2. Нажми **New** → **Web Service**
3. Выбери свой репозиторий `planeta-kassa`
4. Render сам увидит файл `render.yaml` и настроит всё автоматически
5. Нажми **Deploy** и подожди 2-3 минуты

### Шаг 3 — Готово!

Render выдаст ссылку вида `https://planeta-kassa.onrender.com`

Открывай с любого устройства — телефона, планшета, компьютера.

---

## Как запустить локально (для тестирования)

```bash
npm install
npm start
```

Открой в браузере: http://localhost:3000

---

## Что умеет касса

- Кнопки всех услуг: Вход, Жетоны, PlayStation, VR, 7D, 5D
- Выбор кассира
- Счётчики выручки и продаж
- Журнал с временем и кассиром
- Отмена последней продажи
- Отчёт по услугам
- История смен
- Экспорт в CSV (открывается в Excel)
- Автообновление каждые 15 сек (синхронизация между устройствами)

---

## Изменить цены или добавить услугу

Открой файл `public/index.html`, найди массив `SVCS` в начале скрипта и измени нужные значения.

```javascript
{ id:'vhod', name:'Вход в клуб', price:500, icon:'🎪', ... },
```

Измени `price` на нужную цену.
